> Duang

## Install

```
$ npm install duang-cli
$ yarn add duang-cli
```

## Usage

#### 将页面转换为PDF文件
```js
duang p

duang pdf [webUrl...]

duang pdf [webUrl...] -f [fileName] -d [filePath] -t [pdf type]

duang pdf --help
```


```js
duang --help 获得更多帮助
```
